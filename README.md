# school_diary

**Электронный дневник для белорусских школьников**  
Простое и удобное приложение для ведения школьного дневника. Позволяет отслеживать расписание, домашние задания, оценки и посещаемость. Разработано специально для учеников и учителей Беларуси.

## Основные возможности

- Ведение расписания уроков  
- Запись домашних заданий  
- Отслеживание оценок  
- Контроль посещаемости  
- Простой и интуитивный интерфейс  

## Структура проекта

- `active_time` — модуль отслеживания активности  
- `xz` — основной код приложения  
- `xzTests`, `xzUITests` — тесты
